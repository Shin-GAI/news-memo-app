# NewsMemo - 디자인 계획

## 앱 개요
크롬에서 뉴스 기사 링크를 공유받아 AI가 핵심 내용을 요약하고, LinkedIn/SNS용 메모를 자동 생성해주는 Android 앱.

## 브랜드 아이덴티티
- **앱 이름**: NewsMemo
- **슬로건**: 읽은 기사를 인사이트로
- **컬러 팔레트**:
  - Primary: `#1A6BFF` (딥 블루 - 신뢰, 전문성)
  - Accent: `#00C896` (에메랄드 그린 - 공유, 성장)
  - Background Light: `#F8FAFF`
  - Background Dark: `#0F1117`
  - Surface Light: `#FFFFFF`
  - Surface Dark: `#1A1D27`
  - Foreground Light: `#0D1117`
  - Foreground Dark: `#E8ECF4`
  - Muted Light: `#6B7280`
  - Muted Dark: `#9CA3AF`

## 화면 목록

### 1. 홈 화면 (Home)
- **역할**: 저장된 메모 목록 + 앱 사용 안내
- **콘텐츠**:
  - 상단 헤더: 앱 로고 + 제목 "NewsMemo"
  - 빈 상태: 크롬에서 공유하는 방법 안내 (일러스트 + 설명)
  - 메모 목록: FlatList로 저장된 메모 카드 표시
  - 각 카드: 기사 제목, 요약 미리보기, 날짜, 플랫폼 배지(LinkedIn/Twitter/일반)
- **액션**: 카드 탭 → 메모 상세, 스와이프 → 삭제

### 2. 공유 수신 화면 (Share Receive / Processing)
- **역할**: 크롬에서 공유된 링크를 받아 AI 처리
- **콘텐츠**:
  - 링크 미리보기 (URL 표시)
  - 로딩 애니메이션 (AI 분석 중)
  - 진행 단계 표시: "기사 읽는 중..." → "핵심 내용 분석 중..." → "메모 생성 완료"
- **흐름**: 앱이 공유 인텐트로 열림 → 자동으로 AI 처리 시작

### 3. 메모 편집 화면 (Memo Editor)
- **역할**: AI가 생성한 메모를 확인하고 편집
- **콘텐츠**:
  - 원본 기사 링크 (탭하면 브라우저로 열기)
  - AI 생성 요약 (핵심 내용 3-5개 불릿)
  - 플랫폼 선택 탭: LinkedIn / Twitter/X / 일반 텍스트
  - 편집 가능한 메모 텍스트 영역
  - 하단 액션 바: 복사, 공유, 저장 버튼
- **특징**: 플랫폼별로 다른 톤/길이의 메모 자동 생성

### 4. 메모 상세 화면 (Memo Detail)
- **역할**: 저장된 메모 조회 및 재공유
- **콘텐츠**:
  - 기사 정보 (제목, URL, 저장 날짜)
  - 플랫폼별 메모 탭
  - 복사/공유 버튼
  - 삭제 버튼

### 5. 설정 화면 (Settings)
- **역할**: 앱 설정
- **콘텐츠**:
  - AI 요약 언어 설정 (한국어/영어)
  - 기본 공유 플랫폼 설정
  - 저장된 메모 전체 삭제
  - 앱 버전 정보

## 핵심 사용자 흐름

### 플로우 1: 크롬에서 기사 공유
1. 크롬에서 뉴스 기사 읽는 중
2. 공유 버튼 탭 → 공유 시트 열림
3. "NewsMemo"로 공유 선택
4. 앱이 열리며 자동으로 AI 처리 시작
5. 로딩 화면 (3-10초)
6. 메모 편집 화면으로 이동
7. AI 생성 메모 확인 및 편집
8. 플랫폼 선택 후 공유 또는 저장

### 플로우 2: 저장된 메모 재활용
1. 홈 화면에서 메모 카드 탭
2. 메모 상세 화면
3. 복사 또는 공유

## 레이아웃 원칙
- 탭 바: 홈(메모 목록) + 설정 (2탭)
- 공유 인텐트로 열릴 때는 탭 바 없이 전용 처리 화면
- 모든 화면: 한 손 조작 최적화, 중요 액션은 하단 배치
- 카드 디자인: 둥근 모서리(16px), 그림자, 여백 충분히

## 기술 구현 전략
- **Android 공유 인텐트**: `app.config.ts`의 `intentFilters`에 `SEND` 액션 추가 + expo-linking
- **AI 요약**: 서버 tRPC + `invokeLLM` (기사 URL → 웹 스크래핑 → AI 요약)
- **메모 저장**: AsyncStorage (로컬 온디바이스)
- **SNS 공유**: `react-native`의 `Share` API + `expo-clipboard`
- **플랫폼별 메모**: LinkedIn(전문적, 300자), Twitter(간결, 280자), 일반(자유형식)
